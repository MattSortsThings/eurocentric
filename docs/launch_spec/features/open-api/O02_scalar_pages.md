# O02. Scalar pages

This document is part of the [launch specification](../../README.md).

## User story

- **As the Dev**
- **I want** the web application to serve a Scalar documentation page for every OpenAPI document, from an endpoint that allows anonymous requests
- **So that** users can understand how to work with the API release

## Acceptance criteria

**Endpoint...**

- [ ] Succeeds_with_200_OK_and_requested_Scalar_page_for_anonymous_client_when_document_exists
- [ ] Succeeds_with_200_OK_and_empty_Scalar_page_when_document_does_not_exist
