# Eurocentric

**A web API for modelling and analysing the Eurovision Song Contest, 2016-present.**

## Documentation

The [docs/launch_spec](docs/launch_spec/README.md) subdirectory contains the *Eurocentric* launch specification
documents.

The [docs/scraped_data](docs/scraped_data/README.md) subdirectory contains country and contest data scraped from the
official Eurovision website.
